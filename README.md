\# Empanelment Admin Portal



A Laravel-based web application developed for managing the IPA Center of Excellence Expression of Interest Portal.



\## Features

\- Admin Dashboard

\- Authentication System

\- Database Management

\- Laravel Backend

\- Responsive Interface



\## Tech Stack

\- Laravel

\- PHP

\- MySQL

\- XAMPP

\- Bootstrap/CSS

\- JavaScript



\## Installation



```bash

git clone https://github.com/Jesikachoudhary/empanelmentadmin.git

cd empanelmentadmin

composer install

copy .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
